## Appium Cucumberjs more node.js

# Install package
```
npm install
```
# Start Appium Server in background
```
npm run appium
```
# Check Appium-doctor
```
npm run appium-doctor
```

# Run Android Test and generate reports
```
npm run cucumber 
```
