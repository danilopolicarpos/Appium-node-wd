## Appium Cucumberjs more Node.js

# Install package
```
npm install
```
# Start Appium Server in background
```
npm run appium
```
# Check Appium-doctor
```
npm run appium-doctor
```

# Run Android Test and generate reports
```
npm run android_test
```

# Run iOS Test and generate reports
```
npm run ios_test
```
